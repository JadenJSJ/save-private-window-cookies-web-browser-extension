# Changelog

## [2.0.0] - 2026-01-12
- **Major Update**: Extended storage support beyond cookies
- New: localStorage save/restore support
- New: IndexedDB save/restore support
- New: Cache API save/restore support (optional)
- New: Modern popup UI replaces options page
- New: Enable/disable toggle for the extension
- New: Data reconciliation dialog when enabling with active private window
- New: Per-storage-type toggles for granular control
- New: Storage statistics display (total size, origin count)
- New: Backup file format v2 with all storage types (v1 still supported)
- Improved: Restore data on tab navigation within private windows

## [1.3.0] - 2024-07-27
- Manifest updated
- Updated for Firefox first-party isolation
- Options Dark Mode support

## [1.2.0] - 2023-01-18
- The backup file is no longer removed from the web browser's download history
- Updated for Firefox storage partitions
- Manifest V3 update
- Improvements

## [1.1.0] - 2021-09-16
- New: Backup and Restore
- Fixes and improvements

## [1.0.0] - 2021-09-07
- First release
